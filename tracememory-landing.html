SAFETY

Stay on task

Before the run starts, you define what the agent is allowed to do. If it goes off-task, TraceMemory catches it before any damage is done.

TraceMemory monitors tool call sequences against the task contract in real-time. If the agent starts crawling internal docs it shouldn't, we pause the run.

/drift-check /tasks/contract
LEARNING

Autonomous self-improvement

Lessons are signed and stored with full provenance. Future runs retrieve relevant lessons automatically.

Reflect

/api/traces/{id}/reflect

Agent analyzes its own successes and failures after every task completion.

Curate

/api/reflections/{id}/curate

Human-in-the-loop or automated supervisor promotes key insights into Signed Lessons.

Retrieve

/api/lessons/retrieve

Future runs semantically search the lesson database before starting a new task.

EXAMPLE LESSON

"Always verify tool output before calling /api/submit. Previous run failed at step 4 because submit was called with unvalidated data — caused task contract violation."

Injected automatically into next run via POST /api/lessons/retrieve.
